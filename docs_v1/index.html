    Welcome to the official documentation for Digital Assets Management.

    Overview

    The Digital Assets Management system is developed to enhance the efficiency of managing an organization's digital assets. The system aims to:

    • Improve the efficiency of digital asset reporting.
    • Enable users to effectively manage digital asset data.
    • Provide users with easy and comprehensive access to digital asset information.

    System Scope

    This system is designed to manage digital assets within the organization only and does not cover external assets or assets owned by clients.

    Core Functions

    • Add, edit, and delete digital asset information.
    • Search and filter asset data.
    • Export reports in CSV, PDF, and Excel formats.
    • Add, edit, and delete views for saving filtered data.
    • Search views.

    Secondary Functions

    • Register an account.
    • Login with username and password.
    • Login with Google.
    • Edit profile information.
    • Change password.
    • Set access permissions based on user level.
